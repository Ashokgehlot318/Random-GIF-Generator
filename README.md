## Installation
  - Clone the repository to your local machine: <br />
    `git clone https://github.com/Ashokgehlot318/Random-GIF-Generator.git`

  - Install dependencies: <br/>
  
     `cd Random-GIF-Generator` <br/>
    
      `npm install` <br />

      `npm start`
